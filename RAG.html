RAG (Retrieval-Augmented Generation)

Complete tutorial with modern LangChain (2026)

What is RAG?
RAG = Retrieval + Augmented + Generation

Instead of asking an AI model to answer from its own knowledge, we first retrieve relevant information from our own documents, then give that information to the AI to generate the answer.

Example

Suppose you have a PDF about your company.

User asks: "What is the office timing?"

Instead of the LLM guessing, RAG:
1. Searches the PDF
2. Finds: Office timing: Monday-Friday 9:00 AM - 5:00 PM
3. Sends this to the LLM

Prompt becomes:
Context: Office timing: Monday-Friday 9:00 AM - 5:00 PM
Question: What is office timing?

LLM answers: Office timing is Monday to Friday from 9 AM to 5 PM.

Step-by-Step Implementation

1

Load PDF

from langchain_community.document_loaders import PyPDFLoader

loader = PyPDFLoader("documents/book.pdf")
docs = loader.load()
2

Split Documents

from langchain_text_splitters import RecursiveCharacterTextSplitter

text_splitter = RecursiveCharacterTextSplitter(
  chunk_size=1000,
  chunk_overlap=200
)

chunks = text_splitter.split_documents(docs)
3

Create Embeddings

from langchain_huggingface import HuggingFaceEmbeddings

embeddings = HuggingFaceEmbeddings(
  model_name="sentence-transformers/all-MiniLM-L6-v2"
)
4

Create FAISS Database

from langchain_community.vectorstores import FAISS

vector_db = FAISS.from_documents(
  documents=chunks,
  embedding=embeddings
)

vector_db.save_local("faiss_db")
5

Load Database

from langchain_community.vectorstores import FAISS

vector_db = FAISS.load_local(
  "faiss_db",
  embeddings,
  allow_dangerous_deserialization=True
)
6

Connect Groq

from langchain_groq import ChatGroq

llm = ChatGroq(
  model="llama-3.3-70b-versatile",
  api_key="YOUR_GROQ_API_KEY",
  temperature=0
)
7

Create Prompt

from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate.from_template("""
Answer the question using only the context.

Context:
{context}

Question:
{question}
""")
8

Build Chain (LCEL)

from langchain_core.output_parsers import StrOutputParser

chain = (
  prompt
  | llm
  | StrOutputParser()
)
9

Use Retriever

retriever = vector_db.as_retriever(
  search_kwargs={"k": 3}
)

docs = retriever.invoke(question)
10

Build Retrieval Chain

from langchain.chains.combine_documents import create_stuff_documents_chain
from langchain.chains.retrieval import create_retrieval_chain

document_chain = create_stuff_documents_chain(
  llm, prompt
)

retrieval_chain = create_retrieval_chain(
  retriever, document_chain
)

Complete Working Example

# 1. Load PDF
from langchain_community.document_loaders import PyPDFLoader
loader = PyPDFLoader("documents/book.pdf")
docs = loader.load()

# 2. Split Documents
from langchain_text_splitters import RecursiveCharacterTextSplitter
text_splitter = RecursiveCharacterTextSplitter(
  chunk_size=1000,
  chunk_overlap=200
)
chunks = text_splitter.split_documents(docs)

# 3. Create Embeddings
from langchain_huggingface import HuggingFaceEmbeddings
embeddings = HuggingFaceEmbeddings(
  model_name="sentence-transformers/all-MiniLM-L6-v2"
)

# 4. Create FAISS Database
from langchain_community.vectorstores import FAISS
vector_db = FAISS.from_documents(
  documents=chunks,
  embedding=embeddings
)

# 5. Connect Groq
from langchain_groq import ChatGroq
llm = ChatGroq(
  model="llama-3.3-70b-versatile",
  api_key="YOUR_GROQ_API_KEY",
  temperature=0
)

# 6. Create Prompt
from langchain_core.prompts import ChatPromptTemplate
prompt = ChatPromptTemplate.from_template("""
Answer the question using only the context.

Context:
{context}

Question:
{question}
""")

# 7. Create Retriever
retriever = vector_db.as_retriever(
  search_kwargs={"k": 3}
)

# 8. Build Retrieval Chain
from langchain.chains.combine_documents import create_stuff_documents_chain
from langchain.chains.retrieval import create_retrieval_chain

document_chain = create_stuff_documents_chain(
  llm, prompt
)

retrieval_chain = create_retrieval_chain(
  retriever, document_chain
)

# 9. Ask Question
response = retrieval_chain.invoke({
  "input": "What is Artificial Intelligence?"
})

print(response["answer"])

Key Concepts

Embeddings

Numerical vectors that capture semantic meaning. Similar texts have similar vectors.

"Cat" → [0.21, 0.81, 0.13, 0.90, ...]

Similarity Search

Finds the most relevant chunks by comparing vector distances (cosine similarity).

Retriever

Modern LangChain's way to handle document retrieval. Converts vector store into a search interface.
